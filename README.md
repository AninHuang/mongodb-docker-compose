# mongodb-docker-compose
Creating MongoDB standalone, sharding, and replica set using Docker Compose

## Pre-requisites
* [Docker](https://www.docker.com/)

## Example
1) Run init.bat to start the containers
2) Run down.bat to tear the containers down

Notes: To run a batch file from CMD, 
type the path and name of the batch file, and press Enter: 
`C:\PATH\TO\FOLDER\BATCH-NAME.bat.`
